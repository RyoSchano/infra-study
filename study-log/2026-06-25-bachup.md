# backup.sh

## やりたいこと
- 指定したフォルダをアーカイブファイルにしてバックアップする

## 処理
1. バックアップ元を圧縮する
2. アーカイブフォルダに入れる
3. 日時付きでログに記録する 

- 使い方 　
    → backup.sh バックアップ元
    → 引数なしなら使い方を表示

## 設計
```
変数
├ アーカイブフォルダ先 → ARCHIVE_DIR
├ ログファイル先 → LOG_DIR
├ バックアップ元 → SOURCE_DIR
├ ログファイル → LOG_FILE
├ アーカイブファイル → BACKUP_FILE
├ 日時 → DATE
└

引数
├ 
├ 
├ 
└ 
```
