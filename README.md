# Infra Study

問題解決を任せられるような信頼されるインフラエンジニアになるための学習記録です。

## 学習内容
- Ubuntu Server
- Linux
- Shell Script
- Git/GitHub

## ディレクトリ構成
```
infra-study/
├ command-reference/
│           ├ 
│
├ initial-settings/
│           └ 各ツールの設定管理
│
├ project-ideas/
│       ├ create-shell-scrips/
│       ├ 
│
├ runbooks/
│       └ 作成した手順書
│
├ scripts/
│       └ 作成したスクリプト
│
├ study-log/
│       ├ templates/
│       │       └ 学習内容別テンプレ
│       └ 学習記録
└ README
```

## 学習方針
- AIや公式ドキュメントを活用しながら学習
- 手順をそのまま実行するのではなく、設定の目的を理解することを重視
- 可能な限り反復して取り組む
- 障害想定 → 手順書作成 → 自動化
```
    できるだけ
     runbooks/
        ├ disk_usage_alert.md

     scripts/
        ├ disk_usage_check.sh

    のように対応させる。
```
## 課題
- 各ツールの基本操作習得
- 基本コマンドの熟知
- 手順書の作成
- スクリプトの作成
- 英語力
