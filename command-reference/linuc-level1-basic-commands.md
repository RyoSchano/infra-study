# Linux Basic Commands
LinuCレベル1基本コマンド一覧

## Process Management
- ps        現在実行中のプロセスを表示
- top       現在実行中のプロセスを継続的に表示
- pgrep     指定した名前のプロセスに対応するPIDを表示
- kill      PIDでプロセスを終了
- killall   プロセス名指定でプロセスを終了
- pkill     プロセス名指定でプロセスを終了
- jobs      実行中のジョブを参照
- nohup     ログアウト後もプログラムを実行させる

## Service Management
- systemctl     systemdが管理するサービスの起動、停止、状態確認などを行う
- journalctl    systemdのログを閲覧

## File Operation
- ls        ディレクトリ内のファイルを表示、ファイルの属性を表示
- cp        ファイルやディレクトリをコピー
- mv        ファイル・ディレクトリを移動
- mkdir     空のディレクトリを作成
- rm        ファイル・ディレクトリを削除
- rmdir     空のディレクトリを削除
- touch     ファイルのタイムスタンプを変更、空のファイルを作成
- file      ファイルの種別を表示

## File Compression and Archive
- gzip bzip2 xz     圧縮
- gunzip bunzip2    解凍
- tar       複数のファイルをまとめたアーカイブファイルを作成、展開

## File Permissions
- chmod     ファイルのアクセス権を変更
- chown     ファイルの所有者・グループを変更（root権限）
- umask     umask値の設定・確認
- chgrp     ファイルのグループを変更（所属グループ限定）
- ln        ハードリンク・シンボリックリンクを作成

## File Search
- find      指定したディレクトリ以下から検索条件にマッチするファイルやディレクトリを検索
  - 検索式
    - name ファイル名    ファイル名で検索
    - atime 日時        最終アクセス時刻で検索
    - mtime 日時        最終更新時刻で検索
    - perm アクセス権    アクセス権で検索
    - type ファイルの種類 (f, d, l)   ファイルの種類で検索
    - exec コマンド {} \;            マッチしたファイルに対してコマンドを実行
- locate 検索パターン    データベースに基づいてパターン一致ファイルを検索
- updatedb  データベース更新
- which     コマンドを検索して絶対パスを表示
- whereis   コマンドのバイナリファイル、ソースコード、マニュアルファイルを検索
- type      コマンドの情報を表示

## Text Filter
- cat       ファイルの内容を表示
- nl        テキストファイルの一部または全部に行番号を付けて表示
- od        バイナリファイルの内容を8進数や16進数で表示
- head      ファイルの先頭部分を表示
- tail      ファイルの末尾部分を表示
- cut       ファイルの各行から指定したフィールドを取り出す
- paste     1つ以上のファイルを読み込んで行ごとに水平方向に連結
- join      2つのファイルを読み込んで共通のフィールドを持つ行を連結
- expand    テキストファイルの中にあるタブをスペースに変換
- unexpand  行頭にある連続した空白をタブに変換
- fmt       テキストを決められた桁に整形
- pr        ファイルの書式を整える
- tr        標準入力から読み込まれた文字列を変換、削除
- sort      行単位でファイルの内容をソート
- split     指定されたサイズでファイルを分割
- uniq      テキストストリームの中で重複行を1行にまとめる
- wc        ファイルの行数、単語数、文字数を表示
- grep      ファイルやテキストストリームの中に検索文字列があるか調べる
- sed       テキストストリームに対して編集を行う

## Network
- nmcli         NetworkManager を導入したシステムのネットワークの設定などを行う
- ip            ネットワークインターフェースやルーティングテーブルなどを管理する
- ifconfig      ネットワークインターフェース情報を表示・設定
- ifup ifdown   ネットワークインターフェースを有効、無効にする
- ping          指定されたホストにICMPパケットを送り、反応を表示
- traceroute    指定されたホストまでパケットが伝わる経路を表示
- tracepath     指定されたホストまでパケットが伝わる経路を表示
- hostname      ホスト名を表示
- ss            ネットワーク機能に関する様々な情報を表示
- nc            テキストストリームにおけるcatコマンドと同様の働きをネットワーク上で行う
- route         ルーティングテーブルの表示や操作
- host          DNSサーバーを使ってホストやドメインに関する情報を表示
- dig           DNSサーバーに登録されている情報を詳しく表示

## User and Group Management
- useradd       ユーザーアカウントを作成
- usermod       既存のユーザーアカウントを変更
- userdel       ユーザーアカウントの削除
- passwd        パスワード変更（root以外は自分）
- groupadd      グループを作成
- groupmod      既存のグループ情報を変更
- groupdel      グループを削除
- groups        ユーザーが所属しているグループを表示
- id            ユーザーID、グループID、所属グループを表示
- whoami        原罪ログインしているユーザー名を表示

## System Time and Log     
- date          システム時刻を参照して現在の日時を表示
- hwclock       システム時刻をハードウェア時刻にセット
- timedatectl   systemd下で日付と時刻、タイムゾーンを管理
- ntpdate       指定したNTPサーバーから現在時刻を取得
- chronyc       Chrony(NTPサーバー)を管理するコマンド
- logger        ログメッセージを生成
- systemd-cat   コマンドの実行結果をジャーナルに書き込む

## Utilities
- tee           コマンドの実行結果をファイルに保存するとともに画面上にも表示
- xargs         標準入力から受け取った文字列を引数に指定して与えられたコマンドを実行
- dd            入力側に指定したファイルからの入力をファイルもしくは標準出力に送る
- crontab       crontabファイルの編集

## System Information
- free      メモリ使用量を確認
- df        ディスク容量を確認
- du        ファイルやディレクトリのサイズを確認
- uptime    topコマンドの1行目のシステム稼働状況のみ表示
