# tracker.sh

## 概要
作業開始・終了時刻を記録し1日の作業時間を集計するBashスクリプトです。  
日付ごとにログファイルを作成し、作業時間の記録・確認・集計を行うことができます。

## 機能
- 作業開始時間を記録（in）
- 作業終了時間を記録（out）
- 今日のログを表示（show）
- 今日の合計作業時間を表示（summary）
- ログディレクトリの自動生成

## 使い方
``` 
tracker <in|out|show|summary>
```
- 引数なしは使い方表示
