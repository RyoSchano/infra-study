# tracker_v3.sh

## やりたいこと
#### tracker.sh にsummary 機能を追加する  
一日の合計作業時間を引数 summary で表示可能にする

## 設計
1. case に summary を追加
2. 今日のログファイルが存在するか確認
3. in と out の時刻を取り出す  
    1行読む
    ↓
    in なら開始時刻を覚える
    ↓
    次の行読む
    ↓
    out なら差分計算
    ↓
    total に加算
    ↓
    次の行読む

4. 秒に変換して差分を計算
5. 合計秒数を h/m 表示する

## 学んだこと
- test -s ファイルが存在して、かつサイズが0より大きいか
- while read A B C  
    - while 繰り返し
    - read ファイルを一行読む
    - レコードを空白区切りで変数A B C に代入
- < 入力リダイレクト ファイルの内容をSTDINとして渡す
- $(()) → 計算
- 1時間 = 3600秒
- % 余り
