# backup.sh

## やりたいこと
- 指定したディレクトリをアーカイブファイルで保存してログを記録する

## 処理
1. バックアップ元を圧縮する
2. アーカイブフォルダに保存する
3. ログを日時付きで記録する

- 使い方 　
    → backup.sh ファイル名 保存先指定
    → 引数なしは使い方を表示

## 設計
```
変数
├ バックアップ元 → SOURCE_DIR
├ バックアップファイル → BACKUP_FILE
├ 保存先ディレクトリ → ARCHIVE_DIR
├ ログファイル → LOG_FILE
├ ログディレクトリ → LOG_DIR
├ 日時 → DATE
└ スクリプトの場所 → SCRIPT_DIR

引数
├ 
├ 
├ 
└ 
```
