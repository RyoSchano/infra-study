## 2. 使い方・仕様（インタフェース）
```
実行例: tracker.sh < in|out|show|summary >
*引数・入力
├ in        → 開始時刻
├ out       → 終了時刻
├ show      → 今日のログを表示
└ summary   → 今日の作業時間を集計

*保存・出力
├ ログ保存場所: $HOME/tracker-logs/YYYY-MM-DD-tracker.log
└ ログ形式: YYYY-MM-DD HH:MM:SS <in|out>
```
