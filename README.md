# Infra Study

インフラエンジニア転職に向けた学習記録です。

## 学習内容
- Ubuntu Server
- Linux
- Shell Script
- Git/GitHub

## ディレクトリ構成
```
runbooks/
    作成した手順書

scripts/
    作成したスクリプト

study-log/
    templates/
        学習内容別テンプレ

    学習記録
```

## 学習方針
- AIや公式ドキュメントを活用しながら学習
- 手順をそのまま実行するのではなk、設定の目的を理解することを重視
- 可能な限り反復して取り組む
- 障害想定→手順書作成→自動化
```
     runbooks/
        ├ disk_usage_alert.md

     scripts/
        ├ disk_usage_check.sh

    のように対応させること。
```
